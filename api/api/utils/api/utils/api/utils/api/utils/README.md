# 🚀 API de Transformação com IA

API para transformação de imagens usando Inteligência Artificial. Modifique cor de cabelo, tom de pele, cor dos olhos e muito mais!

## ✨ Funcionalidades

- ✅ Detecção facial precisa
- ✅ Transformação de cor de cabelo (8 cores disponíveis)
- ✅ Ajuste de tom de pele (8 tons disponíveis)
- ✅ Modificação de cor dos olhos
- ✅ Suavização de pele
- ✅ Preview rápido das transformações
- ✅ API RESTful documentada

## 🛠️ Tecnologias

- **FastAPI** - Framework web
- **OpenCV** - Processamento de imagens
- **MediaPipe** - Detecção facial
- **NumPy/SciPy** - Operações matemáticas
- **Mangum** - Adaptador para Vercel

## 📋 Pré-requisitos

- Python 3.9+
- Git
- Conta no Vercel (gratuita)
